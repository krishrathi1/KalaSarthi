## ЁЯУЪ Best Practices

### Clear Pronunciation
- Speak clearly and at normal pace
- Pause briefly between commands
- Use complete sentences when possible

### Command Structure
```
Action + Object + Modifier
"Show me cheapest sarees"
"Search for handmade jewelry"
"Filter by textiles category"
```

### Error Recovery
- If misunderstood, rephrase the command
- Use simpler words if complex terms fail
- Try the same command again after a pause

## ЁЯЖШ Troubleshooting

- Verify microphone access

- Check browser compatibility
