## What is Gemini Nano?

Gemini Nano is Google's on-device AI model that runs locally in Chrome browsers, providing real AI responses without internet connectivity. It's perfect for offline use while maintaining privacy.

### Browser Requirements
- **Chrome 127+** (or Chrome Canary/Dev)
- **Desktop or Android** (iOS not supported yet)

### Enable Gemini Nano

1. **Open Chrome Flags**
   - Go to `chrome://flags/`
   - Search for "Prompt API for Gemini Nano"

2. **Enable the Flag**
   - Set "Prompt API for Gemini Nano" to **Enabled**
   - Restart Chrome

3. **Verify Installation**
   - Open Developer Console (F12)
   - Type: `window.ai?.languageModel`
   - Should return an object (not undefined)
